# Задание 1

## Задача 1. List.map

### Текст задачи 

Получить список из сумм цифр натуральных чисел, содержащихся в исходном списке

### Алгоритм решения

Каждое число списка будет подвергаться цикличной обработке посредством деления на 10 с остатком и сложением с другими натуральными цифрами этого числа в следующих циклах, до тех пор пока после деления числа на 10 для следующего цикла оно не станет равно 0. Так будет с каждым числом из списка, с помощью List.map.

# Задача 2


## Задача 1. List.fold

### Текст задачи 

Найти сумму тех элементов списка, которые начинаются на заданную цифру.

### Алгоритм решения

При помощи List.fold смотрим каждый элемент и перед его аккумулированием проверяем с помощью условия на соответствие  с числом заранее указанным пользователем.
